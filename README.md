## ✋🏻 Pré-requisitos

- [Node.js 16.20](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/getting-started)

## :keyboard: Comandos

- Baixar dependências do projeto:

```sh
yarn
// ou
npm install
```

- Realizar build:

```sh
yarn build
// ou
npm run build
```

- Executar projeto:

```sh
yarn dev
yarn server
// ou
npm run dev
npm run server
```
